## 🧩 数据模型

### Project（项目）
```typescript
interface Project {
  id: string;              // 唯一标识 (UUID)
  name: string;            // 项目名称
  description: string;     // 项目描述
  status: ProjectStatus;    // 项目状态
  goal?: ProjectGoal;       // 项目目标（可选）
  path?: string;           // 项目文件夹路径（可选）
  isArchived?: boolean;    // 是否已归档
  createdAt: string;      // 创建时间
  updatedAt: string;      // 更新时间
}

type ProjectStatus = 'planning' | 'in_progress' | 'completed' | 'suspended';
```

### Task（任务）
```typescript
interface Task {
  id: string;              // 唯一标识 (UUID)
  projectId: string;       // 所属项目 ID
  title: string;           // 任务标题
  description: string;     // 任务描述
  status: TaskStatus;      // 任务状态
  priority?: TaskPriority; // 任务优先级（可选）
  tags?: string[];         // 任务标签（可选）
  comments?: TaskComment[];// 任务评论（可选）
  dueDate?: string;        // 截止日期（可选）
  createdAt: string;       // 创建时间
  updatedAt: string;       // 更新时间
}

type TaskStatus = 'todo' | 'in_progress' | 'done';
type TaskPriority = 'low' | 'medium' | 'high';

interface TaskComment {
  id: string;
  content: string;
  createdAt: string;
}
```

### ProjectGoal（项目目标）
```typescript
interface ProjectGoal {
  targetCount: number;    // 目标完成数量
  currentCount: number;    // 当前完成数量
  description: string;    // 目标描述
}
```

### Tag（标签）
```typescript
interface Tag {
  id: string;
  name: string;
  color: string;
  createdAt: string;
}
```

## 🧠 实现逻辑

### 1. 状态管理（Zustand）

**项目 Store (`projectStore.ts`)**
- 使用 Zustand 创建全局项目状态
- 提供 CRUD 操作：`createProject`, `updateProject`, `deleteProject`, `archiveProject`
- 提供筛选方法：`getActiveProjects`, `getArchivedProjects`
- 自动持久化到 localStorage

**任务 Store (`taskStore.ts`)**
- 使用 Zustand 创建全局任务状态
- 提供 CRUD 操作：`createTask`, `updateTask`, `deleteTask`, `toggleTaskStatus`
- 提供统计方法：`getProjectTasks`, `getTaskStats`
- 自动持久化到 localStorage

**标签 Store (`tagStore.ts`)**
- 管理全局标签
- 提供 CRUD 操作：`createTag`, `updateTag`, `deleteTag`
- 自动持久化到 localStorage

### 2. 数据持久化

**localStorage 模式**
- 使用 `src/utils/storage.ts` 中的 `createStorageAdapter` 工厂函数
- 自动保存到 localStorage（key: `task-manager-projects`, `task-manager-tasks`）
- 页面加载时自动恢复数据

**文件系统模式**
- 使用 File System Access API 读写项目文件夹中的 `.task-manager.json`
- 在 `src/utils/fileSystem.ts` 中实现文件系统操作
- 在 `src/components/folder/FolderSync.tsx` 中实现文件夹同步 UI

### 3. 国际化（i18n）

- 使用 `src/i18n/translations.ts` 存储多语言翻译
- 使用 `src/i18n/index.tsx` 中的 `I18nProvider` 提供语言上下文
- 使用 `useI18n` Hook 获取翻译文本

### 4. 主题切换

- 使用 `src/theme/index.tsx` 中的 `ThemeProvider` 提供主题上下文
- 使用 Tailwind CSS 的 `dark:` 类实现暗黑模式
- 主题设置保存到 localStorage

### 5. 拖拽功能（@dnd-kit）

- 使用 `@dnd-kit/core` 的 `DndContext` 提供拖拽上下文
- 使用 `@dnd-kit/sortable` 的 `SortableContext` 实现可排序列表
- 拖拽时自动更新任务状态

### 6. 动画效果（Framer Motion）

- 使用 `AnimatePresence` 实现组件进入/退出动画
- 使用 `motion.div` 实现平滑过渡
- 使用 `layout` 属性实现布局动画

### 7. 性能优化

- 使用 `React.memo` 优化纯组件
- 使用 `useMemo` 缓存计算结果
- 使用 `useCallback` 缓存事件处理函数
- 使用 Vite 的代码分割功能
